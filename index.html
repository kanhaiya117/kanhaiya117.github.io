Cric Score Pro

Knk Ltd. by KK Bharat & Co.

Score every ball. Anywhere.

Cric Score Pro is a cricket scorebook app for local matches, tournaments, school cricket, and friendly games. It helps scorers record runs, wickets, extras, innings, and live match updates with a clean mobile-first experience.

Live scoreboard PDF scorecard Multi-language UI Cloud sync

About the app

Fast scoring

Record 0, 1, 2, 3, 4, 6, wicket, wide, no-ball, bye, and leg-bye events with simple scorer controls.

Live match view

Share a match code so viewers can follow current score, batters, bowler, overs, chase details, and ball-by-ball updates.

Professional reports

Generate an English PDF scorecard with batting, bowling, wickets, innings summary, and top performers.
